# Food Recipe Management System

This is a web-based Food Recipe Management System developed using PHP and MySQL.  
The system allows users to browse food recipes, view categories, and manage favorite recipes easily.

## Features

- User login and authentication
- Browse recipes by categories (Veg, Non-Veg, Egg, Seafood, Jain, Vegan, etc.)
- View detailed recipe pages
- Add recipes to favorites
- Organized category-based navigation
- Database storage using MySQL

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP Server

## How to Run

1. Install XAMPP
2. Copy project folder into htdocs
3. Start Apache and MySQL
4. Import food_recipe.sql into phpMyAdmin
5. Open browser and go to:
   http://localhost/food_recipe
